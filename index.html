Recursos para seguir aprendendo

HTML: Linguaxe de Etiquetaxe de HiperTexto (ou HyperText Markup Language, en inglés) é unha linguaxe de marcas deseñada para crear páxinas web.

CSS: follas de estilo (ou Cascading Style Sheets, en inglés) é unha linguaxe usada para definir a presentación de documentos escritos nunha linguaxe de marcación, como é o caso de HTML ou XML.
